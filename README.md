## Quick Start:
Use your favorite package manager to install the project's dependencies:

```bash
npm: npm install

yarn: yarn

pnpm: pnpm install
```

Run the project:

```bash
npm: npm run start

yarn: yarn start

pnpm: pnpm start
```
